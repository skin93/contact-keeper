# contact-keeper

Simple MERN stack application for creating accounts and keeping contacts. App deployed on Heroku.

## Built with
```
1. HTML5, CSS3, JavaScript ES6, Async/Await, Axios, JSON Web Token
2. React, Context API, React Hooks
3. Node.js, Express
4. MongoDB, mongoose
```
